Password Toogle Demo

You can find here few exemples of password input custom element.

With default style

You can use this custom element without any css framework

Without parameters
<input type="password" is="toggle-password"/>
Override labels
<input type="password" is="toggle-password" data-show-text="Voir" data-hide-text="Cacher"/>
With icons
<input type="password" is="toggle-password" data-display-icon="true"/>

With custom style

You can use this custom element with a custom style

Without parameters
<style>
.custom {
    height: 54px;
    border: 1px solid #dfedff;
    border-radius: 8px;
    outline: none;
    padding: 0 16px;
    color:#498bff;
}
.custom:focus {
    border-color: #498bff;
}
.toggle-password {
    right: 16px !important;
}
.toggle-password:after {
    color:#498bff;
}
</style>
<input type="password" class="custom" is="toggle-password"/>

With bootstrap style

You can also use this custom element with your favorite css framework like boostrap

Without parameters

    <div class="form-group">
        <input type="password" is="toggle-password"/>
    </div>
</form>

<form>
    <div class="form-group">
        <input type="password" is="toggle-password" data-show-text="Voir" data-hide-text="Cacher"/>
